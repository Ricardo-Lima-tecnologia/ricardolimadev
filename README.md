## 🚀 Tecnologias

- **Frontend:** React, TypeScript, TailwindCSS
- **Build:** Vite
- **Deploy:** Netlify (Deploy automático via GitHub)
- **Hospedagem:** https://devricardolima.com

## 💻 Desenvolvimento Local

### Pré-requisitos
- Node.js 18+ 
- npm ou yarn

### Instalação

```bash
# Instalar dependências
npm install

# Iniciar servidor de desenvolvimento
npm run dev

# Build para produção
npm run build
```

O servidor de desenvolvimento estará disponível em `http://localhost:5173`

## 📦 Build

```bash
npm run build
```

Os arquivos de produção serão gerados em `dist/client/`

## 🌐 Deploy

O projeto está configurado com deploy automático via GitHub + Netlify:

1. Faça suas alterações
2. Commit e push para o repositório
3. Netlify detecta automaticamente e faz deploy
